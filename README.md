# flutter_we

一个基于Flutter的情侣日常社交APP，可同时部署在Android和iOS设备上，主要功能有：

- 情侣日常记录
- 备忘录功能
- 情侣账号关联、数据同步
- 数据导出
- 数据与云端备份与恢复
